R Package

autoFRK

The R package, developed using RcppEigen, offers automatic fixed-rank kriging for spatial data, even when data points are irregularly distributed. It leverages a set of basis functions with multi-resolution characteristics, ordered based on their resolutions.

SpatPCA

This R package streamlines regularized principal component analysis by: (1) identifying prominent patterns (eigenfunctions) that exhibit smoothness and localization; (2) facilitating spatial prediction (Kriging) at new locations; (3) accommodating both regularly and irregularly spaced data; and (4) utilizing the efficient alternating direction method of multipliers (ADMM) algorithm.


  • GitHub

SpatMCA

This R package offers regularized maximum covariance analysis by (1) pinpointing smooth and localized patterns that reveal how one spatial process influences another; (2) accommodating both regularly and irregularly spaced data; and (3) employing the efficient alternating direction method of multipliers (ADMM) algorithm..


  • GitHub

influenceAUC

This package focuses on identifying influential observations from the perspective of model diagnostics in binary classification, and provides a couple of visualization tools as well.
